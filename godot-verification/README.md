# 🎮 Godot 源码知识库落地验证

## 概述

使用 Godot Engine 源码作为真实场景，验证各知识库方案的实际效果。

Godot 引擎特点：
- **代码量**: ~200 万行 C++（含 C# 绑定）
- **模块数**: 50+ 核心模块
- **复杂度**: 高（渲染、物理、脚本、编辑器、平台层等）
- **文档**: 相对完善，但代码注释参差不齐

## 验证目标

1. **索引能力** — 能否完整索引 Godot 源码？耗时？资源消耗？
2. **检索质量** — 对代码的语义理解是否准确？跨模块检索效果？
3. **Claude 集成** — 与 Claude Code 配合后，代码理解和生成能力提升多少？
4. **生产可行性** — 部署难度、维护成本、扩展性
